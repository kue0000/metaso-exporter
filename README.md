# Metaso Exporter - 秘塔搜索导出

一个 Chrome 扩展，一键导出 [秘塔AI搜索](https://metaso.cn) 结果为 Markdown 格式。

## 功能特性

- 支持简洁、深入、深度研究三种搜索模式
- 多轮问答完整保留，每轮问题与回答一一对应
- 正文引用自动转为脚注标记 `[^N]`，可跳转至参考文献
- 参考文献保留原始网页超链接
- 来源信息（标题、URL）按轮次归档
- 悬浮导出按钮 + Popup 面板双入口
- 纯前端实现，无需后端服务

## 安装

1. 下载本仓库的 [最新 release](../../releases) 或克隆代码
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目文件夹
5. 访问 metaso.cn 搜索后即可使用

## 使用方式

- **方式一**：点击浏览器工具栏的扩展图标，在 Popup 面板中点击「导出 Markdown」
- **方式二**：在 metaso.cn 搜索结果页，点击右下角悬浮的导出按钮

导出的 Markdown 文件会自动下载到本地。

## 导出格式示例

```markdown
# 村集体收入分那些部分...

> 搜索模式: 简洁（推测）
> 导出时间: 2026/06/11
> 来源: https://metaso.cn/chat/...

---

## 提问（第1轮）: 村集体收入分那些部分...

### AI 回答

村集体收入主要包含四类：经营收入[^1]、投资收益[^2]...

[^1]: [关于规范农村集体'三资'和财务管理的实施意见](http://www.cqwx.gov.cn/...)
[^2]: [村级集体经济年经营性收入](http://www.gaoan.gov.cn/...)

---

## 提问（第2轮）: 集体的东西承包给他人...

### AI 回答

集体资产承包给他人所获收入属于"发包及上交收入"[^1]...

[^1]: [中方县农村集体经济组织财务管理制度](http://www.zhongfang.gov.cn/...)
```
